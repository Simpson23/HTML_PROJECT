3.3 Develop solutions to improve own productivity using IT in digital projects – what would you do differently next time? Team work? Use of Slack?

Next time I would definitely get a team together and delegate tasks related to specific sections to each of them. This would improve efficiency and also a team brings more ideas to the table than an individual. Another benefit of having team members is that when one person is stuck with a problem they can work together to solve it. The amount of time that it would take to finish the project would reduce significantly as there is more than one person working on it so it would at least half the work time. When working with a team communication is vital and without this the team will go through many issues and errors. To resolve this issue I would advise using Slack. This enables the team members to create a channel where they can message each other, share images, hyperlinks or they can even message other team members individually in private chat. When it comes to sharing code I would advise Github as this is dedicated to making code sharing in teams easy.
