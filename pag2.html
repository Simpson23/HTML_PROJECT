1.5 Describing copyright and other constraints affecting websites

Websites can’t take someone else's work and use it as their own as this comes under copyright. This can include videos, pictures, text and even software. If someone wanted to use another individual’s work then they would have to ask for permission. In some cases the copyright owner may even demand some sort of payment to use their work. In the instance that permission is not granted and the website owner still uses the work then they are breaking the law by breaching the copyright act. By breaking this law they are in a position where they can be fined and prosecuted in court.

When it comes to the individuals that may have some sort of disability there could be different struggles that each are going through. 1.7 Describe which are the appropriate filetypes for websites

In order to create a website the programme has to run in a certain version, HTML. We can code HTML in Visual Studio (VS) Code and CSS. In VS Code the base of the website is developed, for example, the title, the header, the images, the text or the sub-titles. In CSS, the websites design and visual attributes are modified, for example, the user can change the colour, add themes or even change the font. Another filetype that may be included in websites are images. These can come in different formats such as .gif, .png or .jpeg. To add in image it must be downloaded and then coded in instead of using a hyperlink in the code, this is so if the website or image gets deleted we can still use it on our website.
